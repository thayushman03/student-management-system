# 🎓 Student Management System

A terminal-based Student Management System built with Python.

## 🔐 Default Login
| Field    | Value     |
|----------|-----------|
| Username | admin     |
| Password | admin123  |

## 🚀 How to Run
```bash
python STUDENT.py
```

## ✨ Features
- Add, Update, Delete, Search students
- Sort records by Name or Marks
- Generate Reports (topper, average, grade distribution)
- Three storage backends: JSON, CSV, SQLite
- Admin login with password hashing
- Export/Import data between formats

## 💾 Storage Backends
| Backend | File          |
|---------|---------------|
| JSON    | students.json |
| CSV     | students.csv  |
| SQLite  | students.db   |
